# To-Do List

A simple JavaScript-based to-do list web app.

## Files

- `index.html` — app structure and UI layout.
- `style.css` — styling for the page.
- `code.js` — JavaScript logic for adding, toggling, and removing tasks.

## Usage

1. Open `index.html` in a browser.
2. Add a new task using the input field.
3. Mark tasks complete or remove tasks as needed.

## Notes

This project is a lightweight client-side to-do list demo.
